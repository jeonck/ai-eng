Markov Chain Monte Carlo (MCMC)

I. 확률 분포의 샘플링과 상태 전이, MCMC 개요​

정의: 직접 샘플링하기 어려운 복잡한 목표 확률 분포로부터, 마르코프 연쇄( Markov Chain )의 정상 분포( Stationary Distribution )를 이용하여 샘플을 추출하는 알고리즘

특징:
( 몬테카를로 ) 난수를 발생시켜 수치적으로 근사 해를 구하는 확률적 방법론
( 마르코프 연쇄 ) 다음 샘플이 오직 현재 샘플에만 의존하며 연쇄적으로 생성되는 구조
( 고차원 처리 ) 수식적으로 적분이 불가능한 고차원 공간의 사후 확률 분포 추정에 탁월

II. MCMC의 주요 알고리즘 및 프로세스​

가. MCMC 샘플링 수행 단계​

  1. 임의의 지점 설정: 초기 샘플 값 결정
  2. 이동 제안: 다음 상태 후보를 확률적으로 제시
  3. 수락 및 거부: 목표 분포와의 비교를 통해 이동 여부 결정
  4. 평형 도달: 초기 값의 영향이 사라지는 Burn-in 기간을 거쳐 목표 분포 수렴

나. 대표적인 MCMC 알고리즘​

알고리즘특징 및 메커니즘비고
Metropolis-Hastings후보 지점을 제안하고 수락 확률에 따라 이동 여부 결정가장 일반적인 형태
Gibbs Sampling다변수 분포에서 각 변수를 조건부 분포에 따라 순차적으로 샘플링다차원 문제 효율적
Hamiltonian MC물리학의 동역학 개념을 도입하여 샘플링 경로의 효율성 증대고차원 수렴 속도 향상

III. MCMC의 활용 분야 및 기술적 고려사항​

항목상세 내용
주요 활용베이지안 통계 추론, 물리 시스템 시뮬레이션, 복잡한 그래픽 모델의 사후 확률 추정
Burn-in초기 상태의 편향을 제거하기 위해 수렴 전 샘플을 버리는 초기 단계
Mixing Rate샘플이 해 공간을 얼마나 빠르고 고르게 탐색하는지를 나타내는 지표

기술 동향: 최근에는 대규모 데이터 처리를 위해 확률적 경사 하강법과 결합된 SGLD (Stochastic Gradient Langevin Dynamics) 등 현대적인 최적화 기법으로 진화하고 있음
