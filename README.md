# AI Engineering

A practical framework for AI engineering, organized into five domains — Infrastructure, Orchestration, Interface, Business Impact, and Governance.

Built with [Hugo](https://gohugo.io/) and the [Hextra](https://github.com/imfing/hextra) theme, deployed to GitHub Pages.

## Local development

```bash
hugo server
```

## Build

```bash
hugo --minify
```

Deployment to GitHub Pages runs automatically via GitHub Actions on every push to `main`.
